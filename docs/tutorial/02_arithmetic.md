# Арифметика и числа

Числовые типы и операции в Funxy.

## Типы данных

| Тип | Описание |
|-----|----------|
| `Int` | 64-битное целое число |
| `Float` | Число с плавающей точкой |
| `BigInt` | Целое произвольной точности |
| `Rational` | Рациональное число |
| `Bool` | Логическое значение |

## Числовые литералы

| Формат | Пример | Тип |
|--------|--------|-----|
| Десятичный | `123`, `-42` | `Int` |
| Шестнадцатеричный | `0xFF` | `Int` |
| Восьмеричный | `0o777` | `Int` |
| Двоичный | `0b101` | `Int` |
| Float | `1.5`, `3.14` | `Float` |
| BigInt | `100n`, `0xFFn` | `BigInt` |
| Rational | `1.5r`, `10r` | `Rational` |

## Операторы

### Арифметические

| Операция | Синтаксис | Примечание |
|----------|-----------|------------|
| Сложение | `a + b` | |
| Вычитание | `a - b` | |
| Умножение | `a * b` | |
| Деление | `a / b` | Int — целочисленное |
| Остаток | `a % b` | |
| Степень | `a ** b` | |

### Сравнения

| Операция | Синтаксис |
|----------|-----------|
| Равно | `a == b` |
| Не равно | `a != b` |
| Меньше | `a < b` |
| Больше | `a > b` |
| Меньше или равно | `a <= b` |
| Больше или равно | `a >= b` |

### Битовые операции (только Int)

| Операция | Синтаксис |
|----------|-----------|
| AND | `a & b` |
| OR | `a \| b` |
| XOR | `a ^ b` |
| NOT | `~a` |
| Сдвиг влево | `a << b` |
| Сдвиг вправо | `a >> b` |

## Примеры

```rust
print(1 + 2)      // 3
print(1.5 + 2.5)  // 4.0
print(2 ** 10)    // 1024
print(~0b101)     // -6
print(7 % 3)      // 1
```

## Встроенные функции

| Функция | Описание |
|---------|----------|
| `print(args...)` | Вывод на stdout |
| `len(collection)` | Длина списка или кортежа |
| `getType(val)` | Тип значения |
| `typeOf(val, Type)` | Проверка типа |
| `panic(msg)` | Прерывание с ошибкой |

```rust
print(len([1, 2, 3]))     // 3
print(getType(42))        // type(Int)

x = 10
if typeOf(x, Int) {
    print("x is Int")
}
```
